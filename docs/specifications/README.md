# Specifications

This directory contains normative implementation specifications for ScratchBird AI integration.

## Structure

- `templates/` - authoring templates
- `drafts/` - in-progress specifications
- `final/` - approved specifications

## Authoring Flow

1. Start from `templates/SPEC_TEMPLATE.md`.
2. Link all supporting sources from `../reference/`.
3. Define exact interfaces, constraints, and acceptance tests.
4. Move to `final/` once reviewed and approved.
