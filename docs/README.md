# Documentation Index

This directory is the canonical documentation root for `ScratchBird-ai`.

## Documentation Sets

- `guides/` - onboarding and operator/developer run guides
- `releases/` - release notes and release-scoped deliverables
- `status/` - current readiness, risks, and milestone status snapshots
- `planning/` - phased implementation backlogs and execution plans
- `specifications/` - normative specs (draft and approved)
- `reference/` - research and external inputs used for specification work
